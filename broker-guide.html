01
Interactive Brokers

הגדרת Flex Query

מה זה Flex Query? זה כמו לבקש מ-IBKR דו"ח של כל הקניות והמכירות שלך. פעם אחת מגדירים — ואחר כך לוחצים Sync ביומן וזהו.
1
כנס לחשבון IBKR
פתח client.ibkr.com והתחבר.
2
פתח את Flex Queries
בתפריט העליון לחץ Performance & Reports ← Flex Queries.
3
צור Flex Query חדש
לחץ Create, בחר Activity Flex Query. תן לו שם כלשהו (למשל Trading Journal) ואשר.
שים לב: קיימים שני סוגים ברשימה — Activity ו-Trade Confirmation. חייבים לבחור Activity.
מצא את Sections → Trades וסמן את 7 השדות האלה:
Symbol טיקר המניה / קריפטו חובה
Open/Close Indicator האם פתיחת פוזיציה או סגירה חובה
Date/Time תאריך ושעת הביצוע חובה
TradePrice מחיר הביצוע בפועל חובה
IB Commission עמלה ששולמה לברוקר חובה
Buy/Sell כיוון העסקה — BUY או SELL חובה
Quantity מספר המניות שבוצעו חובה
פורמט: XML. טווח תאריכים: Last 365 Calendar Days. לחץ Save — זהו, את ה-Query סיימת.
4
העתק את Query ID
חזור לרשימת ה-Flex Queries. ה-Query שיצרת יופיע עם מספר ID — העתק אותו.
Query ID — נראה כך
1462900
5
קבל Flex Token
לך ל-Settings → Flex Queries Delivery (תחת Reporting). שם תמצא את האפשרות לייצר Token — לחץ Generate Token.
שמור את ה-Token עכשיו! IBKR מראה אותו פעם אחת בלבד — אם תסגור את הדף, תצטרך לייצר חדש.
Token — נראה כך
1234567890abcdef1234567890abcdef
6
הזן ביומן והכל מוכן
חזור לאפליקציה → Settings → Interactive Brokers. הדבק את ה-Token ואת ה-Query ID ולחץ Save. אחר כך לחץ Sync.
02
Bybit

יצירת API Key

הרשאות: ה-API Key יהיה Read Only בלבד — היומן רק קורא את ההיסטוריה, לא מבצע פקודות.
1
כנס לחשבון Bybit
פתח bybit.com והתחבר.
2
פתח את API Management
לחץ על תמונת הפרופיל → My Profile → API.
3
צור API Key חדש
לחץ Create New Key, בחר System-generated API Keys.
4
בחר הרשאות
תחת Permissions, סמן רק את שלושת אלה:
  • Trade History — Read
  • Order — Read
  • Position — Read
אל תסמן Withdraw! זה לא נחוץ ועלול להיות מסוכן.
5
שמור את ה-Key וה-Secret
לחץ Submit. Bybit יציג את ה-API Key וה-Secret — שמור אותם עכשיו, הם מופיעים פעם אחת בלבד.

6
הזן ביומן והכל מוכן
חזור לאפליקציה → Settings → Bybit. הדבק את ה-Key וה-Secret ולחץ Save, אחר כך Sync.
03

סנכרון ובדיקה

1
לחץ Sync
לחץ Sync. תוך 10–30 שניות העסקאות יטענו.
2
בדוק שהכל עלה
תראה הודעה כמו "יובאו 23 עסקאות חדשות". אם יש שגיאה — ראה פתרון בעיות למטה.
3
בדוק שהעסקאות מופיעות
לחץ על Stocks או Crypto ובדוק שהעסקאות שם. זהו — סיימת!
היומן רק קורא נתונים מהברוקר — לא שולח פקודות ולא מסחר בשמך.
04

פתרון בעיות נפוצות

"Token לא תקין" או "Unauthorized"
לך ל-IBKR: Manage Account → Settings → Flex Web Service — ודא שהשירות מופעל. אם צריך, לחץ Generate Token מחדש.
"Query ID לא נמצא"
ה-Query ID הוא מספרי בלבד (לדוגמה 1462900). ודא גם שיצרת Activity Flex Query — לא Trade Confirmation.
"0 עסקאות נמצאו" למרות שיש עסקאות
פתח את ה-Flex Query ב-IBKR ובדוק שהטווח הוא Last 365 Calendar Days — אחרת עסקאות ישנות לא יופיעו.
עסקאות מופיעות ב-Stocks אבל ללא Stop Loss
זה נורמלי — IBKR לא שולח פקודות עתידיות, רק עסקאות שבוצעו. הזן Stop Loss ידנית בכל עסקה ביומן.
Bybit — "Invalid API Key"
לך ב-Bybit ל-My Profile → API וודא שה-Key פעיל ולא פג תוקף. אם פג — יצור חדש.
עסקאות כפולות לאחר Sync
רענן את הדף ולחץ Sync שוב — היומן בודק כפילויות ולא ייצור עסקאות חוזרות. אם זה ממשיך, צור קשר.
מוכן? בוא נחבר את הברוקר

כנס לאפליקציה → Settings → בחר את הברוקר שלך ועקוב אחרי המדריך.
